## 筛选策略

### 多因素综合权重
- **技术面 (40%)**: 均线多头、MACD金叉、KDJ信号、量比放大
- **资金流向 (35%)**: 主力净流入、散户净流出、主力占比
- **趋势分析 (25%)**: 振幅控制、换手率适中、避免追高

## 智能筛选算法

### 多因素综合打分算法

系统从A股市场扫描数百支股票，根据以下7个因子实时打分：

| 因子 | 权重 | 评分规则 |
|------|------|----------|
| 量比活跃度 | 15% | 量比>2给15分，>1.5给10分，>1给5分 |
| 涨幅合理性 | 20% | 0%-3%温和上涨20分，3%-6%给15分，6%-9%给10分 |
| 换手率适度 | 15% | 3%-10%换手率适中15分，10%-20%给10分 |
| 价格位置 | 10% | 价格在当日低位(<40%位置)得10分 |
| 振幅稳健 | 10% | 振幅2%-5%得10分，5%-8%得7分 |
| 资金体量 | 15% | 成交额3-15亿得15分，1-3亿得8分 |
| 趋势稳健 | 15% | 开盘涨幅-1%~2%得15分 |

### 风险排除规则

- 排除 ST/*ST 股票
- 排除 科创板 (688开头)
- 排除 涨跌停股票（波动风险太大）
- 排除 换手率 > 25%
- 排除 股价 < 3元或 > 20元

### 算法特点

- **实时扫描**: 每次请求实时获取市场数据
- **动态排名**: 根据当日市场情况动态打分排序
- **透明评分**: 每支股票显示综合评分和推荐理由
- **风险分级**: 根据评分自动划分低/中/高风险
