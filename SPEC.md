# A股短线推荐平台 - 需求与设计规格

## 1. 项目概述

### 1.1 项目名称
**A股短线宝** (A-Share Short-Term Recommender)

### 1.2 目标用户
- 股市新手小白用户
- 对短线交易有兴趣但缺乏经验的投资者
- 需要决策辅助参考的散户

### 1.3 核心价值
为新手用户提供简单易懂的A股短线投资建议，排除科创板高风险品种，聚焦主板和创业板的安全边际较高的股票。

### 1.4 股票范围
- 上海主板 (SH)
- 深圳主板 (SZ)
- 创业板 (CY)
- **排除**：科创板 (KC)

## 4. 筛选策略（多因素综合）

### 4.1 技术面因素 (权重: 40%)

| 指标 | 筛选条件 | 说明 |
|------|----------|------|
| 均线多头 | MA5 > MA10 > MA20 | 短期上涨趋势 |
| MACD | DIF > DEA 且在零轴上方 | 动能增强 |
| KDJ | K值 > D值，K值 < 80 | 处于上升通道但未超买 |
| 成交量 | 今日量比 > 1.5 | 有资金参与 |

### 4.2 资金流向因素 (权重: 35%)

| 指标 | 筛选条件 | 说明 |
|------|----------|------|
| 主力净流入 | > 0 | 主力看多 |
| 主力净流入占比 | > 3% | 资金明显介入 |
| 散户净流出 | > 0 | 筹码集中 |

### 4.3 趋势因素 (权重: 25%)

| 指标 | 筛选条件 | 说明 |
|------|----------|------|
| 涨停次数 | 近5日 < 2次 | 避免追高 |
| 振幅 | < 8% | 走势稳健 |
| 换手率 | 3% ~ 15% | 活跃度适中 |

### 4.4 风险排除规则

- 排除 ST/*ST 股票
- 排除 科创板 (688开头)
- 排除 上市不足60日的新股
- 排除 流通市值 < 50亿 (机构参与度低)
- 排除 换手率 > 25% (炒作风险高)
- **排除 股价 > 20元** (新手友好，低价股更易操作)

## 6. 非功能需求

### 6.1 性能需求
- 首页加载 < 2秒
- 列表页响应 < 1秒
- 数据更新延迟 < 30秒

### 6.2 兼容性
- 响应式设计，支持PC/平板
- 浏览器: Chrome/Firefox/Safari/Edge 最新版

### 6.3 安全需求
- 不存储用户真实交易信息
- 数据仅供参考，不构成投资建议
- 明显的风险提示和免责声明

---

## 7. 风险提示

**郑重声明**：
1. 本平台所有数据仅供参考，不构成任何投资建议
2. 股市有风险，投资需谨慎
3. 短线交易风险极高，可能造成重大损失
4. 新手用户建议先学习基础知识，模拟操作后再入场
5. 请根据自身风险承受能力理性投资
